Fluent PDF is a WordPress plugin that automatically generates PDF documents from [Fluent Forms](https://fluentforms.com) entries. Choose from built-in templates, customize appearance globally or per feed, and deliver PDFs as email attachments or downloadable links — all without writing a single line of code.

For developers, Fluent PDF exposes a PHP API and a full set of WordPress action/filter hooks so you can generate PDFs programmatically, build custom templates, and integrate with any third-party plugin.

## Get started in three steps

<Steps>
  <Step title="Install the plugin">
    Upload and activate **Fluent PDF** from the WordPress admin. See [installation](/installation) for details.
  </Step>
  <Step title="Download fonts">
    Navigate to **Fluent Forms → Add-Ons → Fluent PDF** and click **Download Fonts** to install the required font pack. Then save your global settings.
  </Step>
  <Step title="Create a PDF feed">
    Open any Fluent Form, go to **Settings → PDF Feeds**, and create a new feed. Pick a template, customize the content and appearance, and save. Your PDF will be generated automatically on every form submission.
  </Step>
</Steps>
