<Note>
  **Latest release: v1.1.0** — March 16, 2026. This release integrates Fluent Forms support as a built-in module and includes automatic migration from the legacy `fluentforms-pdf` plugin.
</Note>

## v1.1.0 — March 16, 2026

- Added Fluent Forms integration as a built-in module (no longer requires the separate `fluentforms-pdf` plugin)
- Added automatic migration of PDF feeds and settings from the old `fluentforms-pdf` plugin
- Added plugin coexistence support — both old and new plugins can be active during transition
- Added graceful error handling for missing fonts (friendly error page instead of a fatal PHP crash)
- Updated compatibility with WordPress 6.7
- Updated minimum PHP requirement to 7.1

## v1.0.0 — August 28, 2023

- Added dynamic module system for third-party plugin integrations

## v1.1.7 — June 22, 2023

- Fixed Repeater Field entries not rendering correctly in PDFs
- Fixed Numeric Field formatter entries

## v1.1.6 — October 19, 2022

- Added support for additional languages and character sets

## v1.1.5 — October 16, 2022

- Fixed header image positioning on multi-page PDFs
- Improved invoice template rendering

## v1.1.4 — October 07, 2022

- Improved Invoice template layout and styling

## v1.1.3 — May 19, 2022

- Added public PDF download option (unauthenticated download links via encrypted URLs)

## v1.1.2 — February 24, 2022

- Fixed PDF spacing issue between sections

## v1.1.2 — February 15, 2022

- Added line break support in PDF content
- Added frontend PDF download shortcode for logged-in users after form submission
- Added conditional block support for Fluent Forms Pro (conditionally show/hide PDF body sections)

## v1.1.0 — March 06, 2021

- Updated compatibility with the latest Fluent Forms version
- Added persistent HTML header shown on all PDF pages
- Added dynamic PDF filename support using form field shortcodes
