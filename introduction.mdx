Fluent PDF is a WordPress plugin that turns [Fluent Forms](https://fluentforms.com) entries into PDF documents. When someone submits a form, Fluent PDF can automatically generate a styled PDF, attach it to a notification email, and make it available as a download link — all configured through the WordPress admin.

For developers, Fluent PDF exposes a PHP action hook so you can generate PDFs programmatically from any plugin or theme, and a filter-based API for registering custom templates.

## Supported integrations

Fluent PDF is built around **Fluent Forms** and integrates directly with its form settings, notification system, and submission viewer. All PDF configuration — feeds, templates, email attachments — lives inside the Fluent Forms admin UI.

For other plugins, Fluent PDF exposes a `do_action` hook that any PHP code can call to generate a PDF from arbitrary HTML content. See [Generating PDFs programmatically](/developer/generating-pdfs-programmatically) for details.

## Requirements

| Requirement | Minimum version |
|---|---|
| WordPress | 5.0 |
| PHP | 7.1 |
| Fluent Forms | Any current release |

Your PHP installation must have the following extensions enabled: **mbstring**, **GD**, **DOM**, and **libxml**. Most managed WordPress hosts include these by default. If any extension is missing, the Fluent PDF settings page will show a system status warning with instructions.
